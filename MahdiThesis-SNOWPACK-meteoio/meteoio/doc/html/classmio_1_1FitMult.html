Detailed Description

A class to perform multiple linear regressions.

This class performs linear regressions with multiple predictors. For example, to compute air temperature trends based on elevation, easting, northing (ie predictors). See www.public.iastate.edu/~maitra/stat501/lectures/MultivariateRegression.pdf
