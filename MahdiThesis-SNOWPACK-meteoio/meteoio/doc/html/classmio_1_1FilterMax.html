mio::FilterMax Class Reference
Data processing elements

Detailed Description

Max range filter.

Reject all values greater than the max. Remarks:

  • MAX: the maximum permissible value (in SI, mandatory);
  • SOFT: if set to TRUE, all data larger than the max would be assigned either the maximum permissible value or another value given as an extra argument (optional);
  • MAX_RESET: if SOFT has been set to TRUE, this is the new value for otherwise rejected points (optional).
TA::filter1 = MAX
TA::arg1::MAX = 330
TA::arg1::SOFT = TRUE
TA::arg1::MAX_RESET = 350
